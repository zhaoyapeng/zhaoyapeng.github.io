hexo初使用

本地查看博客

执行如下命令
hexo generate
hexo server

用浏览器打开http://localhost:4000，即可在本地查看

发表文章

执行如下命令
hexo new “文章的名字”

部署至github

执行如下命令
hexo generate
hexo deploy

添加图片

在hexo/source/下创建文件夹image用以存放图片，在需要显示图片的地方添加
