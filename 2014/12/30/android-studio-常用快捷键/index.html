android studio 常用快捷键

以下快捷键适用于mac系统

格式化代码
command + alt +L

对于String资源文件
隐藏R.String.*
command + -

显示R.String.*
command + +

查找当前类中的某个方法
command + alt + o

显示最近编辑的文件
command + E

生成get或者set方法
alt + enter

注释代码（1） //
command + /

注释代码（2）/**/
command + alt + /

预览
fn + f1

生成注释块
/** + 回车
