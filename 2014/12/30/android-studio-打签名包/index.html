android studio 打签名包

打签名包之前，首先要配置签名文件
个人习惯，将keystone放在工程中，如app/keystore文件夹下（keystore文件夹是自己创建的文件）
选择File -> Project Structure 出现如下页面

点击左下角的+添加，并配置 keystore文件的密码和路径

切换栏目至 Build Types
在Signing Config 一栏选择 选择刚刚创建的配置，（注意 左侧的debug和release,debug一栏为配置调试用的keystore, release一栏为正式的keystore）

完成以上步骤，build.gradle中会生成签名配置，也可忽略上述步骤，直接将下边的代码复制进build.gradle文件中

完成签名的配置之后，接下来开始打包，这里我们使用gradle打包，分两步完成
执行gradle clean，清空掉之前生成的build文件夹及其中的文件
执行gradle build，打签名包
完成上述两步骤之后，会在工程下生成build文件夹，app/build/outputs/apk/ 下会有四个apk文件其中*-release.apk是我们所需要的签名包
