一些git常用命令

git remote 写出所有远程主机

git remote -v 列出主机 及地址

git pull origin dev 将远程dev分支与当前分支合并

git branch -r 查看远程分支

git branch -a 查看所有分支（其中带*号为本地主机当前的分支，remotes/主机名/分支名为远程分支（可能会存在多
个）

git branch dev 创建名字为dev的分支

git checkout dev 切换到dev分支

git checkout -b dev 创建dev分支并切换至dev分支

git merge dev 将dev分支合并到当前分支

git branch -b dev 删除dev分支
