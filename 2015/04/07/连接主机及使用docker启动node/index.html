连接主机及使用docker启动node

连接主机

在终端输入 shh 用户名@域名（或者主机ip）点击回车，此时将提示输入密码，争取让输入密码即可连接主机，例如：ssh zhaoyp@zhaoyp.zhhub.com

使用docker启动node

docker restart zhaoyp-web//重启node
docker attach zhaoyp-web//查看日志
