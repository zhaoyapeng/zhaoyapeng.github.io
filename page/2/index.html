node代码保持一直在后台运行

首先需要安装pm2 ,执行npm install pm2 -g

pm2 start ./bin/www — name 工程名字 执行此命令可以使工程一直运行在后台

pm2 restart www 重启node项目

pm2 list 列出 所有使用pm2启动的项目

pm2 logs 项目id 跟踪此项目id的日志
